# プライバシーポリシーテンプレート（日本法ベース）

Webサービス等のプライバシーポリシードラフト作成時に使用するテンプレート。個人情報保護法（2022年改正対応）に基づく構成。`[要確認]` の項目はヒアリングで確認する。

> このテンプレートは参考用のたたき台です。法的な正確性・網羅性は専門家の確認が必要です。GDPR/CCPA対応が必要な場合は追加条項が必要です。

## ヒアリング項目一覧

テンプレート内の `[要確認]` 項目を埋めるために確認が必要な事項:

| 項目 | 質問 |
|------|------|
| サービス名 | サービスの正式名称は？ |
| サービス提供者名 | 会社名・個人名は？ |
| 取得する個人情報 | どのような個人情報を取得しますか？（氏名、メール、電話番号等） |
| 支払い情報 | 決済機能はありますか？決済代行会社を利用しますか？ |
| SNSログイン | SNS連携ログインはありますか？（Google、Facebook等） |
| アクセス解析 | Google Analytics等のアクセス解析ツールを使用しますか？ |
| 広告配信 | 広告配信サービスを利用しますか？ |
| メール配信 | メールマガジン等の配信はありますか？ |
| 海外サーバー | 海外のサーバーやサービスを利用しますか？ |
| GDPR/CCPA | 海外ユーザーへの対応は必要ですか？ |
| 問い合わせ先 | 個人情報に関する問い合わせ先は？ |

## GDPR対応が必要な場合の追加条項

EUユーザーが利用する場合、以下の条項の追加を検討:

- 処理の法的根拠（同意、契約履行、正当利益等）
- データ保護責任者（DPO）の設置
- データポータビリティの権利
- 異議申立ての権利
- 監督機関への苦情申立ての権利
- 個人情報の保持期間

## CCPA対応が必要な場合の追加条項

カリフォルニア州居住者が利用する場合、以下の条項の追加を検討:

- 個人情報のカテゴリと収集ソース
- 個人情報の販売に関するオプトアウトの権利
- 個人情報の削除を要求する権利
- 非差別的対応の保証

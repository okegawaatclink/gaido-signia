# 利用規約テンプレート（日本法ベース）

Webサービス等の利用規約ドラフト作成時に使用するテンプレート。`[要確認]` の項目はヒアリングで確認する。

> このテンプレートは参考用のたたき台です。法的な正確性・網羅性は専門家の確認が必要です。

---

## テンプレート本文

```markdown
# [要確認: サービス名] 利用規約

最終更新日: [要確認: 日付]

## 第1条（適用）

1. 本規約は、[要確認: サービス提供者名]（以下「当社」）が提供するサービス「[要確認: サービス名]」（以下「本サービス」）の利用に関する条件を定めるものです。
2. ユーザーは、本規約に同意した上で本サービスを利用するものとします。

## 第2条（定義）

本規約において使用する用語の定義は以下のとおりです。

1. 「ユーザー」とは、本サービスを利用するすべての方をいいます。
2. 「登録ユーザー」とは、本サービスにアカウント登録を行ったユーザーをいいます。
3. [要確認: サービス固有の定義があれば追加]

## 第3条（アカウント登録）

[要確認: ユーザー登録が必要な場合のみ]

1. ユーザーは、当社所定の方法により登録を行い、アカウントを作成することができます。
2. ユーザーは、正確かつ最新の情報を登録するものとします。
3. アカウントの管理責任はユーザーが負うものとします。

## 第4条（利用料金・支払い）

[要確認: 有料サービスの場合のみ]

1. 本サービスの利用料金は、当社が別途定める料金表に基づきます。
2. 支払方法: [要確認: クレジットカード / 銀行振込 / その他]
3. [要確認: 無料プランの有無、試用期間の有無]

## 第5条（禁止事項）

ユーザーは、本サービスの利用にあたり、以下の行為を行ってはならないものとします。

1. 法令または公序良俗に反する行為
2. 当社または第三者の知的財産権、プライバシー権、名誉権その他の権利を侵害する行為
3. 本サービスのサーバーまたはネットワークに過度の負荷をかける行為
4. 本サービスの運営を妨げる行為
5. 不正アクセスまたはこれを試みる行為
6. 他のユーザーのアカウントを使用する行為
7. 反社会的勢力に対する利益供与その他の協力行為
8. [要確認: サービス固有の禁止事項]

## 第6条（サービスの変更・中断・終了）

1. 当社は、ユーザーへの事前通知なく、本サービスの内容を変更し、または提供を中断もしくは終了することができるものとします。
2. 当社は、本サービスの変更、中断または終了によりユーザーに生じた損害について、一切の責任を負わないものとします。

## 第7条（知的財産権）

1. 本サービスに関する知的財産権は、当社または正当な権利者に帰属します。
2. ユーザーが本サービスを通じて投稿したコンテンツの著作権はユーザーに帰属しますが、当社は本サービスの運営・改善に必要な範囲で当該コンテンツを利用できるものとします。
3. [要確認: UGC（ユーザー生成コンテンツ）がある場合の取扱い]

## 第8条（免責事項）

1. 当社は、本サービスの完全性、正確性、確実性、有用性等について、いかなる保証も行いません。
2. 当社は、ユーザーが本サービスの利用により被った損害について、当社に故意または重大な過失がある場合を除き、一切の責任を負わないものとします。
3. [要確認: 損害賠償の上限額を設定する場合]

## 第9条（個人情報の取扱い）

当社によるユーザーの個人情報の取扱いについては、別途定めるプライバシーポリシーによるものとします。

## 第10条（規約の変更）

1. 当社は、必要と判断した場合には、ユーザーへの事前通知により本規約を変更できるものとします。
2. 変更後の利用規約は、当社が別途定める場合を除き、本サービス上に表示した時点で効力を生じるものとします。

## 第11条（準拠法・管轄裁判所）

1. 本規約の解釈および適用は、日本法に準拠するものとします。
2. 本サービスに関して紛争が生じた場合には、[要確認: 裁判所名]を第一審の専属的合意管轄裁判所とします。

以上

[要確認: サービス提供者名]
制定日: [要確認: 日付]
```

## ヒアリング項目一覧

テンプレート内の `[要確認]` 項目を埋めるために確認が必要な事項:

| 項目 | 質問 |
|------|------|
| サービス名 | サービスの正式名称は？ |
| サービス提供者名 | 会社名・個人名は？ |
| サービス概要 | どのようなサービスですか？ |
| ユーザー登録 | アカウント登録は必要ですか？ |
| 有料/無料 | 利用料金はありますか？ |
| 支払方法 | どのような決済方法を提供しますか？ |
| UGC | ユーザーがコンテンツを投稿する機能はありますか？ |
| 損害賠償上限 | 損害賠償の上限額を設定しますか？ |
| 管轄裁判所 | どの裁判所を管轄としますか？ |
| 固有の禁止事項 | サービス特有の禁止事項はありますか？ |

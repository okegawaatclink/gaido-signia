# 契約書チェックリスト

法務文書レビュー時に確認すべき条項カテゴリの一覧。各カテゴリについて、対象文書に該当する条項があるか、内容に注意すべき点があるかを確認する。

> このチェックリストは参考情報です。契約種別や業種に応じて確認すべき項目は異なります。

## 1. 契約の基本事項

| チェック項目 | 確認ポイント |
|-------------|-------------|
| 契約当事者 | 正式名称・所在地・代表者が正確か |
| 契約目的 | 業務内容・対象範囲が明確か |
| 契約期間 | 開始日・終了日・自動更新条項の有無 |
| 契約金額・支払条件 | 金額・支払期日・支払方法が明記されているか |

## 2. 秘密保持（NDA関連）

| チェック項目 | 確認ポイント |
|-------------|-------------|
| 秘密情報の定義 | 範囲が広すぎないか、例外規定はあるか |
| 秘密保持期間 | 契約終了後の期間は妥当か（一般的には1〜3年） |
| 開示範囲 | 従業員・再委託先への開示条件は明確か |
| 返還・破棄義務 | 契約終了時の秘密情報の取扱いが定められているか |

## 3. 知的財産権

| チェック項目 | 確認ポイント |
|-------------|-------------|
| 成果物の権利帰属 | 著作権・特許権等の帰属先は明確か |
| 既存IP（バックグラウンドIP） | 既存の知的財産の取扱いが定められているか |
| ライセンス条件 | 利用許諾の範囲・条件は明確か |
| 共同開発の場合 | 共同開発成果物の権利配分が明確か |

## 4. 責任制限・損害賠償

| チェック項目 | 確認ポイント |
|-------------|-------------|
| 損害賠償の上限 | 上限金額の設定は妥当か |
| 間接損害・逸失利益 | 免責範囲は明確か |
| 瑕疵担保・契約不適合 | 責任期間・対応範囲は明確か |
| 免責事項 | 不可抗力条項（フォースマジュール）はあるか |

## 5. データ保護・プライバシー

| チェック項目 | 確認ポイント |
|-------------|-------------|
| 個人情報の取扱い | 個人情報保護法に準拠した規定があるか |
| データの所有権 | データの帰属・利用権限は明確か |
| セキュリティ対策 | 技術的・組織的安全管理措置の規定があるか |
| 漏洩時の対応 | 通知義務・対応手順が定められているか |
| 越境移転 | 海外へのデータ移転に関する規定があるか |

## 6. 解除・終了

| チェック項目 | 確認ポイント |
|-------------|-------------|
| 解除事由 | 中途解除の条件は明確か |
| 催告の要否 | 催告なしの即時解除が可能な事由は妥当か |
| 解除時の清算 | 既払金・未払金の精算方法が定められているか |
| 残存条項 | 契約終了後も効力を有する条項が明確か |

## 7. 再委託・第三者

| チェック項目 | 確認ポイント |
|-------------|-------------|
| 再委託の可否 | 事前承認の要否が明確か |
| 再委託先の責任 | 再委託先の行為に対する責任の範囲は明確か |
| 第三者の権利 | 第三者の知的財産権等への配慮があるか |

## 8. 紛争解決

| チェック項目 | 確認ポイント |
|-------------|-------------|
| 準拠法 | 準拠法が明記されているか |
| 管轄裁判所 | 専属的合意管轄が定められているか |
| ADR（裁判外紛争解決） | 調停・仲裁の規定があるか |

## 9. 競業避止・勧誘禁止

| チェック項目 | 確認ポイント |
|-------------|-------------|
| 競業避止期間 | 期間・地域・業種の制限は妥当か |
| 勧誘禁止 | 相手方従業員の勧誘禁止規定の有無 |

## 10. その他

| チェック項目 | 確認ポイント |
|-------------|-------------|
| 反社会的勢力排除 | 反社条項が含まれているか |
| 通知方法 | 通知先・通知方法が明記されているか |
| 契約変更手続 | 変更時の書面合意要件があるか |
| 完全合意条項 | 本契約が唯一の合意であることの確認 |

# コスト見積もりテンプレート

開発・運用にかかるコストを見積もるためのテンプレート。

---

## 見積もり項目

1. 開発コスト（人件費）
2. インフラコスト（クラウド/オンプレ）
3. 外部サービスコスト（API、SaaS）
4. ライセンスコスト

## コスト管理機能

```markdown
| 機能 | 説明 | 実装優先度 |
|------|------|----------|
| 月次コスト上限アラート | 設定閾値超過時に通知 | 必須 |
| 日次利用量ログ | API呼び出し回数・トークン数を記録 | 推奨 |
| コストダッシュボード | コスト推移の可視化 | 任意 |
| 予算予測 | 月末コスト予測 | 任意 |
```

---

## 見直しタイミング

- 月次：実績と見積もりの乖離確認
- 四半期：料金改定の確認、プラン最適化
- 年次：全体見直し

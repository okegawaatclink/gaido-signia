# PBI テンプレート

GitHub Issue作成用のPBIテンプレート。

---

## テンプレート

```markdown
# PBI [Epic番号].[連番]: [ユーザーストーリー形式のタイトル]

## 親Epic

#[Epic Issue番号]

## ユーザーストーリー

**As a** [誰が]
**I want to** [何をしたい]
**So that** [どんな小さな価値が得られる]

## 価値の説明

この PBI が完了すると、ユーザーは具体的に何ができるようになるか：



## INVESTチェック

- [ ] **I**ndependent: 他PBIと独立しているか
- [ ] **N**egotiable: 交渉・調整の余地があるか
- [ ] **V**aluable: ユーザー価値があるか
- [ ] **E**stimable: 見積もり可能か
- [ ] **S**mall: 1スプリントで完了可能か
- [ ] **T**estable: 受入条件が検証可能か

## YAGNIチェック

- [ ] 今この瞬間、これがないと先に進めないか？
- [ ] 作らないことで10倍以上の手戻りが発生するか？
- [ ] 推測ではなく実際の要求に基づいているか？

## 完成の定義（DoD）

- [ ] 受入条件がすべて満たされている
- [ ] コードレビュー完了
- [ ] ユニットテストカバレッジ達成
- [ ] E2Eテスト合格
- [ ] セキュリティレビュー完了

## 受入条件

| # | 条件 | 品質分類 | 回帰テスト |
|---|------|----------|-----------|
| 1 | | [必須]/[推奨]/[任意] | ○/△/× |
| 2 | | | |

## 依存関係

- **前提**:
  - なし
- **後続**:
  - なし

## タスク

- [ ] Task [Epic番号].[PBI連番].[連番]: [タイトル]
- [ ] Task [Epic番号].[PBI連番].[連番]: [タイトル]
```

## 記入例

```markdown
# PBI 1.2: パスワードを忘れても復旧できる

## 親Epic

#1 Epic 1: ユーザーが安全にログインできる

## ユーザーストーリー

**As a** パスワードを忘れたユーザー
**I want to** パスワードをリセットしたい
**So that** アカウントに再度アクセスできる

## 価値の説明

この PBI が完了すると、ユーザーはパスワードを忘れても
メール経由で自力でパスワードを再設定できる。
サポートへの問い合わせを減らし、ユーザー体験を向上させる。

## INVESTチェック

- [x] **I**ndependent: 他PBIと独立しているか
- [x] **N**egotiable: 交渉・調整の余地があるか
- [x] **V**aluable: ユーザー価値があるか
- [x] **E**stimable: 見積もり可能か
- [x] **S**mall: 1スプリントで完了可能か
- [x] **T**estable: 受入条件が検証可能か

## YAGNIチェック

- [x] 今この瞬間、これがないと先に進めないか？ → Yes
- [x] 作らないことで10倍以上の手戻りが発生するか？ → Yes
- [x] 推測ではなく実際の要求に基づいているか？ → Yes

## 完成の定義（DoD）

- [x] 全受入条件が満たされている
- [x] コードレビュー完了
- [x] ユニットテストカバレッジ80%以上
- [x] E2Eテスト合格
- [x] セキュリティレビュー完了

## 受入条件

| # | 条件 | 品質分類 | 回帰テスト |
|---|------|----------|-----------|
| 1 | リセットメールが5分以内に届く | [必須] | ○ |
| 2 | リセットリンクは24時間で無効化される | [必須] | ○ |
| 3 | 新パスワードは過去3回と異なること | [必須] | ○ |
| 4 | リセット完了後に確認メールが届く | [推奨] | △ |

## 依存関係

- **前提**:
  - PBI 1.1
- **後続**:
  - PBI 1.3
  - PBI 1.4

> ※ `build_tasklist.py` 実行時に「PBI X.Y」がGitHub Issue番号に自動解決されます。

## タスク

- [ ] Task 1.2.1: パスワードリセットAPIを実装する
- [ ] Task 1.2.2: リセットメールテンプレートを作成する
- [ ] Task 1.2.3: パスワードリセット画面を作成する
```

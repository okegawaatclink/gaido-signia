## テンプレートの種類

| 種類 | 説明 | 使い方 |
|------|------|--------|
| **テンプレート** | 穴埋め形式 | コピーして項目を埋める |
| **ガイドライン** | 書き方のルール | 参照しながら作成 |
| **リファレンス** | 比較表・選択肢 | 選定時の参考資料 |
